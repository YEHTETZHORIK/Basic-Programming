Programming Basics

Programming သည် ကွန်ပျူတာနှင့် စက်များအတွက် အရေးအသားများကို ရေးသားခြင်းဖြစ်ပြီး အချက်အလက်များကို ထိန်းချုပ်ရန်၊ အလုပ်လုပ်ရန် ရည်ရွယ်သည်။

What is Programming?

Programming သည် ကွန်ပျူတာကို မိမိအလိုအလျောက် အလုပ်လုပ်စေဖို့ ရေးသားသော အစီအစဉ်များဖြစ်သည်။ Programmer များက ကွန်ပျူတာ ဘာလုပ်စေချင်သလဲ ဆိုတာကို Programming Language တွေကို အသုံးပြုပြီး ရေးသားပေးရသည်။

Popular Programming Languages

Python

Python သည် လွယ်ကူပြီး အသုံးပြုရ အဆင်ပြေသော Programming Language တစ်ခုဖြစ်သည်။

အသုံးများသော Application များ - YouTube, Instagram, Spotify

JavaScript

JavaScript သည် Web Development တွင် အဓိက အသုံးပြုသော Programming Language ဖြစ်သည်။

အသုံးများသော Application များ - Facebook, Twitter, LinkedIn

C++

C++ သည် အဆင့်မြင့်စွာ အစီအစဉ်ရေးရန် အသုံးပြုသည်။

အသုံးများသော Application များ - Adobe Products, Microsoft Office, Games (e.g., Counter-Strike)

How Does Programming Work?

Programming သည် အကြီးမားသော Project များကို အသေးစိပ် အစီအစဉ်များအဖြစ် ခွဲခြားပြီး အတူတကွ အလုပ်လုပ်သည်။ Programmer များက အစီအစဉ်များကို ရေးသားပြီး၊ Compiler (သို့) Interpreter များက အစီအစဉ်များကို ကွန်ပျူတာနားလည်သည့် Machine Code အဖြစ် ပြောင်းလဲပေးသည်။

Interesting Facts

  • Programming Languages များသည် ၁၉၅၀ ပြည့်နှစ်များမှ စတင်၍ ဖန်တီးခဲ့သည်။
  • Python သည် ၁၉၉၁ ခုနှစ်တွင် ဖန်တီးခဲ့ပြီး ကျောင်းသားများနှင့် စတင်လာသူများအတွက် အလွန်လွယ်ကူသော Language တစ်ခုဖြစ်သည်။
  • JavaScript သည် အင်တာနက်သုံးစွဲမှုအတွက် အဓိက အသုံးပြုသော Language ဖြစ်သည်။
  • HTML သည် မည်သည့် Programming Language မဆို မဟုတ်ဘဲ၊ စာမျက်နှာများ ဖော်ပြရန် အသုံးပြုသော Markup Language ဖြစ်သည်။
